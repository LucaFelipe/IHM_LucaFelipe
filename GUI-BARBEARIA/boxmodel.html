Demonstração do Box Model

O Box Model do CSS é essencialmente uma caixa que envolve cada elemento do HTML. Ele consiste em: borderm, padding, margin e o conteúdo real.

Esse texto é o conteúdo da caixa. Adicionamos um preenchimento de 50px, uma margem de 20px e uma verde de 15px. Altere as propriedades para entender como elas afetam a renderização desta div
